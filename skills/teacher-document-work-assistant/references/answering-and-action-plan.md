# Answering And Action Plan Rules

Use this when the teacher asks a question after the source inventory or knowledge base exists.

## Answer Shape

```markdown
## 답변

[짧은 결론]

## 근거

- `source path`: 확인한 내용

## 액션플랜

| 순서 | 할 일 | 담당/문의 | 기한 | 근거 |
| --- | --- | --- | --- | --- |

## 확인할 질문

- [부족한 근거 또는 담당 부서에 물어볼 질문]
```

## Rules

- Start from `knowledge-base/work-knowledge-base.md`.
- Open raw files only to verify missing or disputed evidence.
- Use exact dates from sources. If there is no exact date, write "문서상 기한 확인 필요".
- Distinguish "해야 한다" from "작년에 이렇게 했다".
- Include a first action that the teacher can do immediately.

## Save Action Plans

Save reusable answers to `action-plans/YYYY-MM-DD-topic.md`.

Use a short Korean topic slug, for example:

- `action-plans/2026-03-이번달-우선순위.md`
- `action-plans/2026-04-작년-서식-확인.md`
- `action-plans/2026-05-매뉴얼-작년방식-차이.md`

If an action plan already exists for the same question, append an "업데이트" section with the current date instead of overwriting useful history.
