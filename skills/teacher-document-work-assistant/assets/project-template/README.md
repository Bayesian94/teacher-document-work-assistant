# 업무 비서 작업 폴더

이 폴더는 문서 기반 업무 비서를 만들기 위한 빈 작업 폴더입니다.

실제 전임자 문서, 교육청 매뉴얼, 공문, 서식, 운영 결과 자료는 사용자가 직접 `raw/` 폴더에 넣어야 합니다. 이 폴더 구조만으로는 업무 지식베이스를 만들 수 없습니다.

## 폴더

```text
raw/                 실제 원자료를 넣는 곳
selected/            분류 후 참조하기 쉽게 바로가기를 정리하는 곳
knowledge-base/      자료 목록과 업무 지식베이스
handover-package/    새 담당자용 인수인계 패키지
action-plans/        질문별 액션플랜 누적
assistant-rules.md   업무 비서의 응답 원칙
```

## 시작 방법

1. 이 폴더를 새 업무명으로 복사합니다.
2. 실제 문서를 `raw/`에 넣습니다.
3. Codex에 "teacher-document-work-assistant Skill로 이 폴더의 업무 비서를 만들어줘"라고 요청합니다.
